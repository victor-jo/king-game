# 🎯 AimGuard

> **앱을 열고 싶다면, 먼저 게임을 클리어하세요.**

AimGuard는 macOS 백그라운드에서 실행되며, 지정한 앱(카카오톡, 디스코드, 유튜브 등)을 실행하려 할 때 **랜덤 미니게임**을 제시하는 생산성 잠금 도구입니다.

## 동작 원리

```
앱 실행 시도
     ↓
프로세스 감지 (1초 간격 폴링)
     ↓
즉시 강제 종료
     ↓
랜덤 미니게임 실행 (5종 중 가중치 추첨)
     ↓
    ┌──────────────┐
  성공              실패 / 포기
    ↓                   ↓
앱 정상 실행       게임 재시도 또는 복귀
```

감시 대상 앱을 열면 해당 프로세스를 즉시 종료하고 미니게임 화면으로 전환합니다. 게임을 클리어해야만 앱이 실행됩니다.

---

## 게임 종류

총 5종의 미니게임이 **가중치 기반 덱 시스템**으로 선택됩니다.
10회마다 모든 게임이 최소 1회 이상 등장하도록 설계되어 있습니다.

| 게임 | 비중 | 핵심 능력 |
|------|------|-----------|
| 🎯 에임 게임 | 10% | 정확성·순발력 |
| 🐛 벌레잡기 | 10% | 반응속도 |
| ⌨️ 타자 게임 | 30% | 타이핑 속도·정확도 |
| 🏋️ 모션 게임 | 20% | 신체 운동 |
| 🔊 소리 게임 | 30% | 목청·소음 |

---

### 🎯 에임 게임

화면에 번호가 매겨진 원형 타겟이 무작위로 배치됩니다.
**1번부터 순서대로** 클릭해야 하며, 잘못된 타겟을 클릭하면 즉시 실패합니다.

- **클리어 조건**: 제한 시간 내 모든 타겟을 순서대로 클릭
- **실패 조건**: 순서 오클릭 또는 시간 초과
- **설정값**: 타겟 수 (3~13개), 제한 시간 (5~30초)

---

### 🐛 벌레잡기

4종류의 벌레가 화면을 돌아다닙니다. 6종류의 무기 중 랜덤으로 지급되며, 목표 점수를 채워야 합니다.

| 벌레 | 속도 | 점수 | 특징 |
|------|------|------|------|
| 🐛 애벌레 | 느림 | 10점 | 직선 이동 |
| 🪳 바퀴벌레 | 빠름 | 30점 | 방향 랜덤 변경 |
| 🐜 개미 | 보통 | 20점 | 무리 이동 |
| 🕷️ 거미 | 보통 | 50점 | HP 2, 대각선 이동 |

| 무기 | 데미지 | 방식 |
|------|--------|------|
| 🔨 망치 | 3 | 클릭 |
| 🪚 전기톱 | 2 | 드래그 |
| 🔫 기관총 | 1 | 빠른 연사 |
| 🔥 화염방사기 | 1 | 드래그 스트림 |
| 💧 수압기 | 2 | 광역 폭발 |
| ⚡ 레이저 | 5 | 정밀 클릭 |

- **클리어 조건**: 제한 시간 내 목표 점수 달성
- **설정값**: 목표 점수 (100~500점), 제한 시간 (10~60초)

---

### ⌨️ 타자 게임

20개의 명언 풀에서 3개의 문장이 랜덤으로 제시됩니다. 각 문장을 입력하고 Enter로 제출합니다.

- **평가 기준**:
  - 정확도 (기본값 80% 이상)
  - 타속 KPM (한국어 두벌식 기준 평균 500 이상)
- **클리어 조건**: 3문장 모두 정확도·타속 기준 충족
- **설정값**: 정확도 기준 (60~100%), 제한 시간 (15~60초)

---

### 🏋️ 모션 게임

웹캠과 **MediaPipe 포즈 인식**을 사용하여 실제 운동을 인식합니다.
3종목 중 하나가 랜덤으로 선택됩니다.

| 종목 | 인식 기준 |
|------|-----------|
| 🏋️ 스쿼트 | 무릎 각도 75° 이하로 내려갔다가 168° 이상으로 복귀 |
| 💪 푸쉬업 | 팔꿈치 각도 75° 이하로 구부렸다가 168° 이상으로 복귀 |
| 🧘 싯업 | 상체-골반-무릎 각도 150° 이상 누웠다가 45° 이하로 일어남 |

- 화면에 실시간 카메라 피드와 골격 오버레이 표시
- 카메라 또는 MediaPipe 사용 불가 시 자동으로 다른 게임으로 폴백
- **설정값**: 목표 횟수 (3~15회), 제한 시간 (20~90초)

---

### 🔊 소리 게임

마이크로 소리를 입력받아 dB 임계값 이상의 소음을 **5초간 연속 유지**해야 합니다.

- dB 레벨 바와 유지 시간 바가 실시간으로 표시됨
- 소리가 끊기면 유지 시간이 초기화됨
- **설정값**: 목표 dB (80/90/100dB), 제한 시간 (20~45초)

## 주요 기능

### 백그라운드 실행 & 시스템 트레이

앱은 시작 시 창 없이 트레이 아이콘으로만 존재합니다.
트레이 아이콘을 클릭하면 설정 화면을 열 수 있습니다.

### 감시 대상 앱 관리

설치된 모든 앱이 목록으로 표시됩니다. 토글로 **화이트리스트(감시 제외)**를 설정합니다.

- **ON (초록)**: 감시 제외 — 자유롭게 실행 가능
- **OFF (회색)**: 감시 대상 — 게임 클리어 후 실행

Finder, Safari, Terminal 등 macOS 시스템 앱은 기본적으로 감시 제외됩니다.

### 가중치 덱 시스템

`랜덤`이지만 `공평한` 게임 선택 구조입니다.

```
덱 구성 (10개):
bug × 1, aim × 1, keyboard × 3, motion × 2, audio × 3
→ 셔플 후 순서대로 소비
→ 소진 시 자동 재생성
```

10번 플레이하면 모든 게임 종류가 최소 1회 이상 등장합니다.

### 절차적 사운드 생성

WAV 파일 없이 사운드를 코드로 생성합니다.
망치 타격음, 레이저 효과음, 성공/실패 멜로디 등이 수학 함수로 만들어집니다.

## 설치 및 실행

### 개발 환경

```bash
# 가상환경 생성 및 활성화
python3 -m venv .venv
source .venv/bin/activate

# 의존성 설치
pip install -r requirements.txt

# 실행
python main.py
```

### macOS 앱 번들 빌드

```bash
pip install py2app
python setup.py py2app
# dist/AimGuard.app 생성됨
```

> **권한 안내**: 카메라(모션 게임), 마이크(소리 게임) 권한이 필요합니다.
> macOS 시스템 환경설정 → 개인 정보 보호 및 보안에서 허용해주세요.

## 기술 스택

| 분류 | 라이브러리 |
|------|-----------|
| GUI | PySide6 (Qt 6) |
| 프로세스 감시 | psutil |
| 모션 인식 | MediaPipe Tasks API + OpenCV |
| 오디오 입력 | sounddevice |
| macOS 카메라 열거 | AVFoundation (PyObjC) |
| 앱 번들 | py2app |
