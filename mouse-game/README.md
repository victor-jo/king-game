# 🎯 AimGuard

에임 게임 기반 앱 잠금 프로그램 — 특정 프로그램을 실행하려면 미니 게임을 클리어해야 합니다!

## 개요

카카오톡, 디스코드 등 지정된 프로그램이 실행되면 자동으로 감지하고, 미니 게임을 표시합니다.
게임을 클리어해야만 해당 프로그램이 실행됩니다.

## 🎮 게임 모드

### 🎯 에임 게임 (순차 클릭)
번호가 적힌 타겟을 1번부터 순서대로 클릭! 잘못된 순서 → 실패.

### 🪲 벌레 잡기 (BugSmasher)
6종 무기(🔨🪚🔫🔥💧⚡)로 벌레를 잡아 목표 점수 달성!

## 주요 기능

- **프로세스 감지** — 잠금 설정된 프로그램 실행 시 자동 감지 + 종료
- **게임 선택** — 에임 게임 또는 벌레 잡기 선택 가능
- **설정 관리** — 프로그램별 잠금 ON/OFF, 게임별 난이도 조절
- **시스템 트레이** — 백그라운드 동작, 트레이 아이콘 클릭으로 설정 접근
- **다크 모드 UI** — 세련된 다크 테마 인터페이스

## 설치 및 실행

```bash
# 가상환경 생성 + 활성화
python3 -m venv venv
source venv/bin/activate

# 의존성 설치
pip install -r requirements.txt

# 실행
python main.py
```

## 사용법

1. 설정 화면에서 잠금할 프로그램의 토글을 **ON**으로 설정
2. 게임 타입 선택 (에임 게임 / 벌레 잡기)
3. 난이도 설정 (타겟 수, 제한 시간, 목표 점수 등)
4. **감시 시작** 클릭
5. 잠금된 프로그램 실행 시 → 게임 표시 → 클리어 시 프로그램 실행!

## 프로젝트 구조

```
├── main.py              # 앱 진입점
├── main_window.py       # 메인 설정 화면 + 시스템 트레이
├── aim_game.py          # 에임 게임 위젯 (순차 클릭)
├── bug_game.py          # 벌레 잡기 게임 위젯 (6종 무기)
├── process_monitor.py   # 프로세스 감시 스레드 (QThread)
├── config.py            # JSON 설정 관리
├── requirements.txt     # 의존성
└── docs/
    └── spec.md          # 프로그램 명세서
```

- **PySide6** 6.10+ (Qt for Python)
