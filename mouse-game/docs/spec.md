# 🎯 AimGuard — 앱 잠금 게임

## 개요

특정 프로그램(카카오톡, 디스코드 등)이 실행되면 자동으로 감지하고, 미니 게임을 클리어해야만 해당 프로그램을 실행할 수 있게 합니다.

## 게임 1: 에임 게임 (순차 클릭)

- 번호가 적힌 원형 타겟을 순서대로 클릭
- 잘못된 순서로 클릭하면 실패
- 제한 시간 내 클리어 필요

## 게임 2: 벌레 잡기 (BugSmasher)

- 화면에 나타나는 벌레를 6종 무기로 잡기
- 제한 시간 내 목표 점수 달성 시 성공

### 무기

| 무기 | 공격 방식 | 쿨다운 |
|------|----------|--------|
| 🔨 망치 | 클릭 지점 타격 | 1.0초 |
| 🪚 전기톱 | 드래그 경로 절단 | 없음 |
| 🔫 기관총 | 연속 클릭 발사 | 0.1초 |
| 🔥 화염방사기 | 드래그 방향 화염 | 없음 |
| 💧 수압기 | 클릭 → 원형 범위 공격 | 1.5초 |
| ⚡ 레이저 | 클릭 2점 → 직선 관통 | 2.0초 |

### 벌레

| 벌레 | HP | 점수 | 특징 |
|------|-----|------|------|
| 🐛 애벌레 | 1 | 10 | 직선 이동 |
| 🪳 바퀴벌레 | 1 | 30 | 불규칙 이동 |
| 🐜 개미 | 1 | 20 | 그룹 이동 |
| 🕷️ 거미 | 2 | 50 | 대각선 이동 |

## 기능 목록

### 메인 화면 (설정)
- [x] 설치된 프로그램 목록 표시 + 잠금 ON/OFF 토글
- [x] 게임 선택 (에임 게임 / 벌레 잡기)
- [x] 게임별 난이도 설정
- [x] 설정 저장/불러오기 (JSON 파일)

### 프로세스 감지
- [x] 백그라운드에서 주기적으로 프로세스 확인
- [x] 새로 실행된 프로세스만 감지 (기존 실행 중인 것은 무시)
- [x] 감지 시 프로세스 종료 → 게임 표시
- [x] 게임 성공 시 프로세스 허용 (종료될 때까지 재감지 안 함)

### 시스템 트레이
- [x] 시스템 트레이 아이콘으로 최소화
- [x] 트레이 메뉴에서 설정 열기 / 종료

## 기술 요구사항

- Python 3.14+
- PySide6 6.10+
- `psutil` — 프로세스 감지용
- `subprocess` — 프로그램 실행용
- JSON — 설정 저장용
